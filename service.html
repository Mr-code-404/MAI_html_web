Roadmap:

  • 前端：

    1、web基础： 2、编程语言： JavaScript 基础 • CSS 基础 3. 进阶（W3School, MDN) • HTML: 表格、表单、多媒体、访问性、语义化 • JavaScript: 作用域、闭包、对象原型 • 学习并了解 Bootstrap 和 jQuery，并用它来写个协会官网 • 使用浏览器的开发者工具来调试网站

  • 现代性前端入门：

    1、基础： • Web技术 • 了解 ECMAScript，异步编程，Promise（阮一峰教程、MDN） • 了解 SASS， SCSS， LESS（官方文档） 2. 工具入门 （官方文档，百度） • NPM, Yarn, Webpack • Gulp 3. 框架入门 • 从 jQuery 过渡到 Vue3（官方文档） • 从 Bootstrap 过渡到 Antd（官方文档）

  • 资源推荐：（按需使用即可）：

    1. 大部分内容： https://www.bilibili.com/video/BV1Bb411v7w8 2. MDN权威入门: https://developer.mozilla.org/zh-CN/docs/Learn 3. 作品参考（功能、UI）: https://codepen.io/ 4. 阮一峰的ES6教程： https://es6.ruanyifeng.com/ 5. NPM, Webpack, Vue3: https://www.bilibili.com/video/BV1Pz4y1S7Uv
