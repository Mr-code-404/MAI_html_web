Changing Font

Wer uses 3 Font family: 'Heebo'(Default) & 'Poppins' & 'Montserrat' from the Google Fonts Library. To change the Font, you will need to Edit the Bottom Link with your Custom Font link

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

    @import url('https://fonts.googleapis.com/css?family=Heebo:400,700&display=swap');
    @import url('https://fonts.googleapis.com/css?family=Poppins:400,500,700&display=swap');
    @import url('https://fonts.googleapis.com/css?family=Montserrat:600,700&display=swap');

Custom JS coding standard:

    $(window).on('scroll', function () {
        if (!$('#menu').hasClass('open')) {
            if ($(document).scrollTop() > 70) {
                $('header').addClass('fixed_header');
            } else {
                $('header').removeClass('fixed_header');
            }
        }else{
            $('header').addClass('fixed_header');
        }
    });

    function smoothScroolInit() {
        $('a').smoothScroll({
            speed: 1000
        });
    }
